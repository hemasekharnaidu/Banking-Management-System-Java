# Banking Management System

A console-based banking application developed using Java that simulates basic banking operations such as account creation, deposit, withdrawal, and balance inquiry.

## Features
- Create new bank accounts
- Deposit money
- Withdraw money
- Check account balance
- Simple and user-friendly console interface

## Technologies Used
- Java
- OOP Concepts
- JDBC
- MySQL

## How to Run
1. Clone the repository
2. Compile the program
3. Run the Java file

```bash
javac BankingManagementSystem.java
java BankingManagementSystem
